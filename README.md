# Pelota Paleta - Defensores de Santos Lugares

Aplicación para registrar partidos de pelota paleta, con autenticación anónima de Firebase, almacenamiento de datos en Firestore, y una interfaz para gestionar jugadores, puntuaciones y resúmenes diarios.

## Instalación

1. Clona el repositorio.
2. Instala las dependencias con `npm install`.
3. Configura las variables de entorno para Firebase (ver `.env`).
4. Ejecuta la app con `npm start`.

## Despliegue

Para desplegar en Vercel:
1. Sube el proyecto a un repositorio de GitHub.
2. Conecta el repositorio a Vercel y despliega.
